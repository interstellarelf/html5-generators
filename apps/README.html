Animation-Station

Our plan:

1) replicate the animation-styles found in modern 2d games

2) create an easy-to-use web-service for creating animations from these styles.

What is the best coa for building the app?
  1. Define a modular design and build on a framework.
  2. Skin the app entirely and make it web-functional.
  3. Create modules and forms side by side.
  4. Define a Cli and build process + testing for the animation product.
  5. Implement payment system for the app.
  6. Document the entire system into acceptance testing phase.
  7. Commit to acceptance testing and write a business proposal.
